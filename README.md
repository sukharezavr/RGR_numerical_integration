# Численное интегрирование методами Ньютона-Котеса

Программа для вычисления определённых интегралов с автоматическим подбором шага.

## Реализованные методы

- **Прямоугольники** (средние) — порядок точности 2
- **Трапеции** — порядок точности 2
- **Симпсон** — порядок точности 4 (требует чётного n)
- **Три восьмых** — порядок точности 4 (требует n кратного 3)

## Тестовые функции

- \(x^2\) на \([0,1]\) (многочлен)
- \(e^{-x^2}\) на \([-2,2]\) (интеграл Пуассона)
- \(\sqrt{|x|}\) на \([-1,1]\) (разрыв производной)

## Требования
- Python 3.x
- mathplotlib

python main.py
