# Численное интегрирование

Программа для вычисления определённых интегралов методами Ньютона-Котеса:
- Средние прямоугольники (порядок точности 2)
- Трапеции (порядок точности 2)
- Симпсон (порядок точности 4)
- Три восьмых (порядок точности 4)

## Состав
- `integration.py` — реализация квадратурных формул
- `runge.py` — правило Рунге и адаптивный подбор шага
- `tests.py` — тестовые функции (x², e^{-x²}, корень)
- `main.py` — запуск исследований

## Использование
```bash
python RGR_integr.py
